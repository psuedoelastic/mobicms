УСТАНОВКА:

1) В случае необходимости, установите права 777 на папки:
   /system/cache/
   /system/logs/
   /system/config/
   /uploads/ (на все подпапки)

2) Создайте базу данных и залейте в нее install.sql

3) Скопируйте файл database.php в /system/config/ предварительно прописав в него реквизиты доступа к базе данных

4) Система готова к работе. LOGIN: admin PASSWORD: admin

ВНИМАНИЕ!
В каждой ветке репозитория находятся свои актуальные SQL файлы.
При переключении на другую ветку необходимо:
- удалить из базы данных все таблицы
- залить все SQL файлы, которые находятся в папке /install/ выбранной ветки.
