## The main possibilities of the system:
- PSR-0 compliant core framework
- modular architecture
- a high level of security
- multilingual
- and many other things...

## System Requirements
- PHP 5.5 +
- MySQL (InnoDB support required) 5.1 +
- .htaccess support
